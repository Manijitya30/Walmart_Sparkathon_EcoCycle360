### Setup

1. **Backend**
   - Install dependencies: `cd backend && npm install`
   - Configure `.env` with your database credentials.
   - Start server: `npm start`

2. **ML API**
   - Install dependencies: `cd ML_api && pip install -r requirements.txt`
   - Start Flask server: `python app.py`

3. **Admin & Customer Frontends**
   - Install dependencies: `cd admin && npm install` and `cd customer && npm install`
   - Start dev servers: `npm run dev` in each directory
